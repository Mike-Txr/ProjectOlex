**Daddy's First Christmas** is a turn-based role-playing game developed in Python using the Arcade framework.

This school project serves as a **prototype** for a larger game concept, and many features, mechanics, and story elements are not yet implemented.

## Gameplay

The game combines exploration with turn-based RPG combat.

### Overworld

Players can travel through different areas of the game world, interact with characters and enemies, and progress through the story.

### Combat System

Battles take place in a turn-based combat system where players must choose their actions carefully to defeat enemies and survive encounters.

Available combat mechanics include:

* **Standard-Attack**
* **Power-Attacks**
* **Items**
* **Blocking**

The more you fight, the stronger you will get.

### Character Stats

The player character uses several attributes during gameplay, including:

* **Health** – Determines how much damage the player can withstand.
* **Power** – Used for stronger special attacks.
* **Attack** – Influences the damage dealt to enemies.

Each level-up gives you the opportunity to increase one of these stats.
